# Process-Scheduler
A program written in Java that demonstrates three process scheduling algorithms

Write	a	program	(Java	or	C) for	the	implementation	of	the	following	Process	Scheduling	Algorithm.	
1. FCFS	
2. SJF	
3. Round	Robin
